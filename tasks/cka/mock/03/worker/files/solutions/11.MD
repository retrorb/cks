## Task 11: Create Ingress

```bash
kubectl config use-context cluster1-admin@cluster1
```

```yaml
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: main-ingress
  namespace: ingress-ns
spec:
  ingressClassName: nginx
  rules:
  - host: app.k8s.local
    http:
      paths:
      - path: /
        pathType: Prefix
        backend:
          service:
            name: ingress-app-svc
            port:
              number: 80
```

```bash
kubectl apply -f main-ingress.yaml

# Verify
kubectl get ingress main-ingress -n ingress-ns
```

**Key concepts:**
- `ingressClassName: nginx` binds the Ingress to the nginx-ingress-controller
- The `host` field routes by hostname (requires DNS or /etc/hosts entry)
- `pathType: Prefix` matches the path and all sub-paths
