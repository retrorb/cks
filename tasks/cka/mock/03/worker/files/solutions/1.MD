## Task 1: Create ClusterIP Service

```bash
kubectl config use-context cluster1-admin@cluster1

kubectl expose deployment backend -n app-1 \
  --name=backend-service \
  --port=80 \
  --target-port=80 \
  --type=ClusterIP
```

Or using a manifest:

```yaml
apiVersion: v1
kind: Service
metadata:
  name: backend-service
  namespace: app-1
spec:
  type: ClusterIP
  selector:
    app: backend
  ports:
  - port: 80
    targetPort: 80
```

**Key concepts:**
- ClusterIP is the default service type — internal cluster traffic only
- The selector must match the pod labels of the target deployment
- Port 80 is the service port; targetPort is the container port
