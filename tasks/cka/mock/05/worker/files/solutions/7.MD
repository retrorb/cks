## Task 7: Save kubectl top pods Output

```bash
kubectl config use-context cluster1-admin@cluster1

# Wait for metrics-server to be ready (if needed)
kubectl get pods -n kube-system | grep metrics-server

# Save top pods output
mkdir -p /var/work/tests/artifacts/7
kubectl top pods -n monitor-ns --sort-by=cpu --context cluster1-admin@cluster1 \
  > /var/work/tests/artifacts/7/top-pods.txt

# Verify
cat /var/work/tests/artifacts/7/top-pods.txt
```

**Key concepts:**
- `kubectl top` requires metrics-server to be running
- `--sort-by=cpu` sorts output by CPU usage
- Output shows: NAME, CPU(cores), MEMORY(bytes)
