## Task 14: Fix Ingress Backend Port

```bash
kubectl config use-context cluster1-admin@cluster1

# Edit the ingress to fix backend port
kubectl edit ingress broken-ingress -n ingress-fix
# Change: number: 9090 -> number: 80

# Or use patch
kubectl patch ingress broken-ingress -n ingress-fix --type='json' \
  -p='[{"op":"replace","path":"/spec/rules/0/http/paths/0/backend/service/port/number","value":80}]'

# Verify
kubectl get ingress broken-ingress -n ingress-fix -o yaml
```

**Key concepts:**
- Ingress backend port must match the Service port (not container port)
- Wrong port causes 502 Bad Gateway errors
- Check service ports with `kubectl get svc -n ingress-fix`
