## Task 19: Fix Pod nodeSelector

```bash
kubectl config use-context cluster1-admin@cluster1

# Check why pod is pending
kubectl describe pod stuck-pod -n affinity-ns

# Delete and recreate without nodeSelector
kubectl delete pod stuck-pod -n affinity-ns

cat << 'EOF' | kubectl apply -f -
apiVersion: v1
kind: Pod
metadata:
  name: stuck-pod
  namespace: affinity-ns
spec:
  containers:
  - name: stuck-pod
    image: nginx:alpine
    ports:
    - containerPort: 80
EOF

# Verify
kubectl get pod stuck-pod -n affinity-ns
```

**Key concepts:**
- `nodeSelector` constrains which nodes a pod can be scheduled on
- If no nodes match the selector, pod stays Pending indefinitely
- Check node labels with `kubectl get nodes --show-labels`
