## Task 16: Remove NoExecute Taint and Uncordon

```bash
kubectl config use-context cluster1-admin@cluster1

# Find node with maintenance taint
kubectl describe nodes | grep -B5 'maintenance=true'

# Get the node name
NODE=$(kubectl get nodes -o json | jq -r '.items[] | select(.spec.taints[]?.key=="maintenance") | .metadata.name' 2>/dev/null)
# Or manually identify: kubectl get nodes

# Remove the taint
kubectl taint node $NODE maintenance=true:NoExecute-

# Uncordon if needed
kubectl uncordon $NODE

# Verify
kubectl describe node $NODE | grep Taints
kubectl get nodes
```

**Key concepts:**
- `NoExecute` taint evicts running pods without matching tolerations
- Remove taints by appending `-` to the taint key:value:effect
- Always uncordon after removing maintenance taints
