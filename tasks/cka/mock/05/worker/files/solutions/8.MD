## Task 8: Save kubectl top nodes Output

```bash
kubectl config use-context cluster1-admin@cluster1

# Save top nodes output
mkdir -p /var/work/tests/artifacts/8
kubectl top nodes --context cluster1-admin@cluster1 \
  > /var/work/tests/artifacts/8/top-nodes.txt

# Verify
cat /var/work/tests/artifacts/8/top-nodes.txt
```

**Key concepts:**
- `kubectl top nodes` shows resource usage across all nodes
- Output: NAME, CPU(cores), CPU%, MEMORY(bytes), MEMORY%
- Requires metrics-server to be running in kube-system
