## Task 10: Save Sidecar Container Logs

```bash
kubectl config use-context cluster1-admin@cluster1

# Save sidecar container logs (specify container with -c)
mkdir -p /var/work/tests/artifacts/10
kubectl logs multi-pod -n multi-ns -c sidecar --context cluster1-admin@cluster1 \
  > /var/work/tests/artifacts/10/sidecar.txt

# Verify
cat /var/work/tests/artifacts/10/sidecar.txt

# List containers in the pod
kubectl get pod multi-pod -n multi-ns -o jsonpath='{.spec.containers[*].name}'
```

**Key concepts:**
- For multi-container pods, specify the container with `-c <container-name>`
- Without `-c`, logs from the first container are shown
- Use `kubectl describe pod` to see all container names
