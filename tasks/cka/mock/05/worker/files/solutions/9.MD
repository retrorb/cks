## Task 9: Save Pod Logs

```bash
kubectl config use-context cluster1-admin@cluster1

# Save last 50 lines of log-pod logs
mkdir -p /var/work/tests/artifacts/9
kubectl logs log-pod -n log-ns --tail=50 --context cluster1-admin@cluster1 \
  > /var/work/tests/artifacts/9/logs.txt

# Verify
cat /var/work/tests/artifacts/9/logs.txt
```

**Key concepts:**
- `kubectl logs <pod>` retrieves container logs
- `--tail=N` limits to the last N lines
- For streaming logs use `-f` (follow)
