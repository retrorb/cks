# Task 13 — Create StorageClass fast-sc + PVC fast-pvc + Pod storage-pod (5 pts)

## Solution

Create StorageClass `fast-sc`:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: storage.k8s.io/v1
kind: StorageClass
metadata:
  name: fast-sc
provisioner: kubernetes.io/no-provisioner
volumeBindingMode: WaitForFirstConsumer
EOF
```

Create PVC `fast-pvc` in namespace `storage-ns`:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: fast-pvc
  namespace: storage-ns
spec:
  storageClassName: fast-sc
  accessModes:
  - ReadWriteOnce
  resources:
    requests:
      storage: 2Gi
EOF
```

Create Pod `storage-pod` that mounts the PVC at `/data`:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: v1
kind: Pod
metadata:
  name: storage-pod
  namespace: storage-ns
spec:
  containers:
  - name: storage-pod
    image: nginx:alpine
    volumeMounts:
    - mountPath: /data
      name: fast-storage
  volumes:
  - name: fast-storage
    persistentVolumeClaim:
      claimName: fast-pvc
EOF
```

Verify:
```bash
kubectl get sc fast-sc --context cluster1-admin@cluster1
kubectl get pvc fast-pvc -n storage-ns --context cluster1-admin@cluster1
kubectl get pod storage-pod -n storage-ns --context cluster1-admin@cluster1
```
