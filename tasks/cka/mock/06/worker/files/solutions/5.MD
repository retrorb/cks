# Task 5 — Fix bad-image-app deployment image in image-ns (4 pts)

## Problem
Deployment `bad-image-app` in namespace `image-ns` uses image `nginx:doesnotexist999` which cannot be pulled.

## Solution

```bash
kubectl set image deployment/bad-image-app bad-image-app=nginx:alpine \
  -n image-ns --context cluster1-admin@cluster1
```

Verify:
```bash
kubectl rollout status deployment/bad-image-app -n image-ns --context cluster1-admin@cluster1
kubectl get pods -n image-ns --context cluster1-admin@cluster1
```
