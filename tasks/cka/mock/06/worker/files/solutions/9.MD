# Task 9 — Remove broken=true:NoSchedule taint from worker node (4 pts)

## Problem
A worker node in cluster1 has taint `broken=true:NoSchedule`, preventing the `taint-test` deployment from scheduling pods on it.

## Solution

Find the tainted node:
```bash
kubectl get nodes --context cluster1-admin@cluster1 -o custom-columns=NAME:.metadata.name,TAINTS:.spec.taints
```

Remove the taint:
```bash
NODE=$(kubectl get nodes -l work_type=infra_core --context cluster1-admin@cluster1 \
  -o jsonpath='{range .items[*]}{.metadata.name} {.spec.taints}{"\n"}{end}' | grep 'broken' | awk '{print $1}')
kubectl taint node $NODE broken=true:NoSchedule- --context cluster1-admin@cluster1
```

Verify:
```bash
kubectl get nodes --context cluster1-admin@cluster1
kubectl get pods -n default -l app=taint-test --context cluster1-admin@cluster1
```
