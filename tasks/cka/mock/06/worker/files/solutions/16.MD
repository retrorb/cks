# Task 16 — Change retain-pv reclaimPolicy to Retain + save to file (4 pts)

## Problem
PV `retain-pv` has `persistentVolumeReclaimPolicy: Delete`. Change it to `Retain` and save the value to `/var/work/tests/artifacts/16/policy.txt`.

## Solution

Change the reclaimPolicy:
```bash
kubectl patch pv retain-pv --context cluster1-admin@cluster1 \
  --type='json' -p='[{"op":"replace","path":"/spec/persistentVolumeReclaimPolicy","value":"Retain"}]'
```

Save the policy value to file:
```bash
kubectl get pv retain-pv --context cluster1-admin@cluster1 \
  -o jsonpath='{.spec.persistentVolumeReclaimPolicy}' \
  > /var/work/tests/artifacts/16/policy.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/16/policy.txt
kubectl get pv retain-pv --context cluster1-admin@cluster1 \
  -o jsonpath='{.spec.persistentVolumeReclaimPolicy}'
```
