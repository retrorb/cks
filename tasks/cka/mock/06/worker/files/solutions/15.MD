# Task 15 — Create PV local-pv + PVC local-pvc in pv-ns (4 pts)

## Solution

Create PersistentVolume `local-pv`:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: v1
kind: PersistentVolume
metadata:
  name: local-pv
spec:
  capacity:
    storage: 5Gi
  accessModes:
  - ReadWriteOnce
  persistentVolumeReclaimPolicy: Retain
  storageClassName: local-storage
  hostPath:
    path: /mnt/data
EOF
```

Create PVC `local-pvc` in namespace `pv-ns` that binds to it:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: local-pvc
  namespace: pv-ns
spec:
  storageClassName: local-storage
  accessModes:
  - ReadWriteOnce
  resources:
    requests:
      storage: 5Gi
EOF
```

Verify:
```bash
kubectl get pv local-pv --context cluster1-admin@cluster1
kubectl get pvc local-pvc -n pv-ns --context cluster1-admin@cluster1
```
