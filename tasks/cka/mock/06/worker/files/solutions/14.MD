# Task 14 — Create StorageClass missing-sc to fix stuck-pvc in pvc-ns (4 pts)

## Problem
PVC `stuck-pvc` in namespace `pvc-ns` is in Pending state because StorageClass `missing-sc` does not exist.

## Solution

Create the missing StorageClass:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: storage.k8s.io/v1
kind: StorageClass
metadata:
  name: missing-sc
provisioner: kubernetes.io/no-provisioner
volumeBindingMode: WaitForFirstConsumer
EOF
```

Verify:
```bash
kubectl get storageclass missing-sc --context cluster1-admin@cluster1
kubectl get pvc stuck-pvc -n pvc-ns --context cluster1-admin@cluster1
```
