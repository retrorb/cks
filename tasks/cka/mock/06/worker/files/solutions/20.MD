# Task 20 — Create ConfigMap + Secret + Pod + save env output (8 pts)

## Solution

Create ConfigMap `app-cm`:
```bash
kubectl create configmap app-cm \
  --from-literal=DB_HOST=prod-db \
  -n config-app-ns \
  --context cluster1-admin@cluster1
```

Create Secret `app-secret`:
```bash
kubectl create secret generic app-secret \
  --from-literal=DB_PASS=s3cr3t \
  -n config-app-ns \
  --context cluster1-admin@cluster1
```

Create Pod `config-pod` with both as environment variables:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: v1
kind: Pod
metadata:
  name: config-pod
  namespace: config-app-ns
spec:
  containers:
  - name: config-pod
    image: nginx:alpine
    envFrom:
    - configMapRef:
        name: app-cm
    - secretRef:
        name: app-secret
EOF
```

Wait for pod to be Running, then save env output:
```bash
kubectl wait pod config-pod -n config-app-ns --context cluster1-admin@cluster1 \
  --for=condition=Ready --timeout=60s

kubectl exec config-pod -n config-app-ns --context cluster1-admin@cluster1 -- env \
  > /var/work/tests/artifacts/20/env.txt
```

Verify:
```bash
grep DB_HOST /var/work/tests/artifacts/20/env.txt
grep DB_PASS /var/work/tests/artifacts/20/env.txt
```
