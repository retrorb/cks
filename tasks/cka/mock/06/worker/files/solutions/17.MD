# Task 17 — Rolling update web-app to nginx:alpine, rollback, save image (6 pts)

## Solution

Rolling update to `nginx:alpine`:
```bash
kubectl set image deployment/web-app web-app=nginx:alpine \
  -n rollout-ns --context cluster1-admin@cluster1

kubectl rollout status deployment/web-app -n rollout-ns --context cluster1-admin@cluster1
```

Rollback to previous version (`nginx:1.25`):
```bash
kubectl rollout undo deployment/web-app -n rollout-ns --context cluster1-admin@cluster1
kubectl rollout status deployment/web-app -n rollout-ns --context cluster1-admin@cluster1
```

Save current image to file:
```bash
kubectl get deployment web-app -n rollout-ns --context cluster1-admin@cluster1 \
  -o jsonpath='{.spec.template.spec.containers[0].image}' \
  > /var/work/tests/artifacts/17/image.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/17/image.txt
# Expected: nginx:1.25
kubectl rollout history deployment/web-app -n rollout-ns --context cluster1-admin@cluster1
```
