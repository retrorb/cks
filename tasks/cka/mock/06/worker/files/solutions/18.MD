# Task 18 — Create HPA scalable-app-hpa for scalable-app in hpa-ns (5 pts)

## Solution

```bash
kubectl autoscale deployment scalable-app \
  --name=scalable-app-hpa \
  -n hpa-ns \
  --context cluster1-admin@cluster1 \
  --min=1 --max=5 --cpu-percent=50
```

Or declaratively:
```bash
kubectl apply --context cluster1-admin@cluster1 -f - <<EOF
apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: scalable-app-hpa
  namespace: hpa-ns
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: scalable-app
  minReplicas: 1
  maxReplicas: 5
  metrics:
  - type: Resource
    resource:
      name: cpu
      target:
        type: Utilization
        averageUtilization: 50
EOF
```

Verify:
```bash
kubectl get hpa scalable-app-hpa -n hpa-ns --context cluster1-admin@cluster1
```
