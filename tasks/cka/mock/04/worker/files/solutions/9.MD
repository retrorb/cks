## Task 9: Check Certificate Expiration

```bash
kubectl config use-context cluster1-admin@cluster1

# SSH to master and run kubeadm certs check-expiration
MASTER=$(kubectl get nodes -o jsonpath='{.items[?(@.metadata.labels.node-role\.kubernetes\.io/control-plane)].status.addresses[0].address}')

mkdir -p /var/work/tests/artifacts/9

ssh -o StrictHostKeyChecking=no ubuntu@$MASTER \
  "sudo kubeadm certs check-expiration" \
  > /var/work/tests/artifacts/9/certs.txt

# View the output
cat /var/work/tests/artifacts/9/certs.txt
```

**Key concepts:**
- `kubeadm certs check-expiration` shows certificate expiry dates
- Certificates should be renewed before expiry using `kubeadm certs renew`
- Common certs: apiserver, apiserver-kubelet-client, front-proxy-client, etcd
