## Task 6: Drain and Uncordon Node

```bash
kubectl config use-context cluster1-admin@cluster1

# Get the worker node name
NODE=$(kubectl get nodes -l work_type=infra_core -o jsonpath='{.items[0].name}')

# Drain the node
kubectl drain $NODE \
  --ignore-daemonsets \
  --delete-emptydir-data

# Uncordon to make schedulable again
kubectl uncordon $NODE
```

**Key concepts:**
- `drain` evicts pods and cordons the node
- `--ignore-daemonsets` skips DaemonSet pods (they can't be moved)
- `--delete-emptydir-data` removes pods with emptyDir volumes
- `uncordon` removes the NoSchedule taint added by drain
