## Task 3: Fix Broken RoleBinding

```bash
kubectl config use-context cluster1-admin@cluster1

# View the current broken RoleBinding
kubectl get rolebinding broken-rb -n fix-rbac -o yaml

# Delete and recreate with correct SA
kubectl delete rolebinding broken-rb -n fix-rbac

kubectl create rolebinding broken-rb -n fix-rbac \
  --role=fix-role \
  --serviceaccount=fix-rbac:fix-sa
```

**Key concepts:**
- RoleBinding subjects cannot be edited in place (immutable)
- Delete and recreate to fix the subject reference
- RoleRef is also immutable once created
