## Task 10: Install Helm Release from Local Chart

```bash
# Install myapp from local chart
helm install myapp /var/work/helm/myapp \
  --namespace helm-ns

# Verify
helm list -n helm-ns
kubectl get pods -n helm-ns
```

**Key concepts:**
- `helm install <release-name> <chart-path>` installs from a local chart directory
- `--namespace` specifies the target namespace
- `helm list` shows installed releases
