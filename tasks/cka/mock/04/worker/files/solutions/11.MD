## Task 11: Upgrade Helm Release

```bash
# Upgrade my-release with replicaCount=3
helm upgrade my-release bitnami/nginx \
  --namespace helm-upgrade-ns \
  --set replicaCount=3

# Verify
helm list -n helm-upgrade-ns
kubectl get deployment -n helm-upgrade-ns
```

**Key concepts:**
- `helm upgrade <release> <chart>` upgrades an existing release
- `--set key=value` overrides specific chart values
- Revision number increments with each upgrade
