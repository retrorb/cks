## Task 7: Generate Bootstrap Token

```bash
kubectl config use-context cluster1-admin@cluster1

# SSH to master node and generate token
MASTER=$(kubectl get nodes -o jsonpath='{.items[?(@.metadata.labels.node-role\.kubernetes\.io/control-plane)].status.addresses[0].address}')

ssh -o StrictHostKeyChecking=no ubuntu@$MASTER \
  "sudo kubeadm token create" > /var/work/tests/artifacts/7/token.txt

# Verify
cat /var/work/tests/artifacts/7/token.txt
```

**Key concepts:**
- Bootstrap tokens follow the format `[a-z0-9]{6}.[a-z0-9]{16}`
- Tokens expire by default after 24 hours
- `kubeadm token list` shows existing tokens
