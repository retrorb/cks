## Task 17: Create StorageClass and PVC

```bash
kubectl config use-context cluster1-admin@cluster1

# Create StorageClass
cat << 'EOF' | kubectl apply -f -
apiVersion: storage.k8s.io/v1
kind: StorageClass
metadata:
  name: local-sc
provisioner: kubernetes.io/no-provisioner
volumeBindingMode: WaitForFirstConsumer
EOF

# Create PVC
cat << 'EOF' | kubectl apply -f -
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: local-pvc
  namespace: storage-ns
spec:
  storageClassName: local-sc
  accessModes:
  - ReadWriteOnce
  resources:
    requests:
      storage: 1Gi
EOF

# Verify
kubectl get sc local-sc
kubectl get pvc local-pvc -n storage-ns
```

**Key concepts:**
- `no-provisioner` means PVs must be created manually
- `WaitForFirstConsumer` delays binding until a pod uses the PVC
- This StorageClass is suitable for local storage scenarios
