## Task 12: Rollback Helm Release

```bash
# Rollback my-release to revision 1
helm rollback my-release 1 \
  --namespace helm-upgrade-ns

# Verify
helm list -n helm-upgrade-ns
helm history my-release -n helm-upgrade-ns
```

**Key concepts:**
- `helm rollback <release> <revision>` rolls back to a specific revision
- A rollback creates a new revision (e.g., revision 3 that mirrors revision 1)
- `helm history` shows all revisions
