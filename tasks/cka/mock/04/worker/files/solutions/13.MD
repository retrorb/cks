## Task 13: Apply Kustomize Configuration

```bash
# Apply kustomize from task13 directory
kubectl apply -k /var/work/kustomize/task13/

# Verify
kubectl get deployment kustomize-app -n kustomize-ns
```

**Key concepts:**
- `kubectl apply -k <directory>` applies a kustomization
- The `kustomization.yaml` file defines resources and transformations
- Kustomize allows customizing YAML without modifying original files
