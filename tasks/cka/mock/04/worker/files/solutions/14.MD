## Task 14: Create Kustomize Overlay

```bash
# Create overlay kustomization.yaml with image patch
cat > /var/work/kustomize/task14/overlay/kustomization.yaml << 'EOF'
apiVersion: kustomize.config.k8s.io/v1beta1
kind: Kustomization
resources:
- ../base
images:
- name: nginx
  newTag: alpine
EOF

# Apply the overlay
kubectl apply -k /var/work/kustomize/task14/overlay/

# Verify
kubectl get deployment base-deploy -n overlay-ns -o jsonpath='{.spec.template.spec.containers[0].image}'
```

**Key concepts:**
- Overlays extend bases by adding patches or transformations
- `images` field can change image names and tags
- Base resources are referenced relatively from the overlay
