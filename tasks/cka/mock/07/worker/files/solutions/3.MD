# Task 3 — CRD/CR explain and create prod-monitor (5 pts)

## Problem
Save `kubectl explain monitor.spec` output and create a Custom Resource `prod-monitor` in `monitor-ns`.

## Solution

```bash
# Save explain output
kubectl explain monitor.spec > /var/work/tests/artifacts/3/explain.txt

# Create the CR
cat <<EOF | kubectl apply -f -
apiVersion: observability.example.com/v1
kind: Monitor
metadata:
  name: prod-monitor
  namespace: monitor-ns
spec:
  interval: "30s"
  target: "api-server"
EOF
```

Verify:
```bash
cat /var/work/tests/artifacts/3/explain.txt
kubectl get monitor prod-monitor -n monitor-ns -o yaml
```
