# Task 5 — Sidecar log-aggregator pod (5 pts)

## Problem
Create a pod with main container and sidecar sharing a volume for log aggregation.

## Solution

```bash
cat <<EOF | kubectl apply -f -
apiVersion: v1
kind: Pod
metadata:
  name: log-aggregator
  namespace: sidecar-ns
spec:
  containers:
  - name: app
    image: nginx:alpine
    volumeMounts:
    - name: log-vol
      mountPath: /var/log/app
  - name: log-collector
    image: busybox
    command: ["sh", "-c", "tail -f /var/log/app/access.log"]
    volumeMounts:
    - name: log-vol
      mountPath: /var/log/app
  volumes:
  - name: log-vol
    emptyDir: {}
EOF
```

Verify:
```bash
kubectl get pod log-aggregator -n sidecar-ns
kubectl get pod log-aggregator -n sidecar-ns -o jsonpath='{.spec.containers[*].name}'
kubectl get pod log-aggregator -n sidecar-ns -o jsonpath='{.spec.volumes[*].name}'
```
