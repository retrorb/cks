# Task 8 — Fix heavy-pod requests to fit quota (5 pts)

## Problem
Pod `heavy-pod` is Pending because resource requests exceed the ResourceQuota. Reduce requests to fit.

## Solution

```bash
# Delete the pending pod
kubectl delete pod heavy-pod -n quota-ns

# Recreate with reduced resources
cat <<EOF | kubectl apply -f -
apiVersion: v1
kind: Pod
metadata:
  name: heavy-pod
  namespace: quota-ns
spec:
  containers:
  - name: heavy
    image: nginx:alpine
    resources:
      requests:
        cpu: "200m"
        memory: "256Mi"
      limits:
        cpu: "200m"
        memory: "256Mi"
EOF
```

Verify:
```bash
kubectl get pod heavy-pod -n quota-ns
kubectl get resourcequota compute-quota -n quota-ns
```
