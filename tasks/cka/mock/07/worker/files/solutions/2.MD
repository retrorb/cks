# Task 2 — CNI plugin name and pod CIDR (3 pts)

## Problem
Identify the CNI plugin and cluster pod CIDR, save to artifact files.

## Solution

```bash
# Identify CNI plugin (check /etc/cni/net.d/ on a node, or look at running pods)
kubectl get pods -n kube-system | grep -iE 'calico|cilium|flannel|weave'
echo "calico" > /var/work/tests/artifacts/2/cni-name.txt

# Get pod CIDR from kubeadm-config ConfigMap
kubectl get cm kubeadm-config -n kube-system -o jsonpath='{.data.ClusterConfiguration}' | grep podSubnet
# or
kubectl get cm kubeadm-config -n kube-system -o yaml | grep podSubnet
echo "10.0.0.0/16" > /var/work/tests/artifacts/2/pod-cidr.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/2/cni-name.txt
cat /var/work/tests/artifacts/2/pod-cidr.txt
```
