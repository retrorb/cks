# Task 1 — Helm install analytics chart (4 pts)

## Problem
Install the local Helm chart at `/var/work/helm/analytics/` as release `analytics` in namespace `helm-install-ns` with custom image tag and service port.

## Solution

```bash
helm install analytics /var/work/helm/analytics/ \
  -n helm-install-ns \
  --set image.tag=v2.1,service.port=9090
```

Verify:
```bash
helm list -n helm-install-ns
kubectl get deploy analytics -n helm-install-ns -o jsonpath='{.spec.template.spec.containers[0].image}'
kubectl get svc analytics-svc -n helm-install-ns -o jsonpath='{.spec.ports[0].port}'
```
