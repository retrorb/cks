# Task 4 — Kubelet inspect (4 pts)

## Problem
SSH to a worker node and extract kubelet config, cgroup driver, and containerd SystemdCgroup value.

## Solution

```bash
# Get worker node IP
WORKER=$(kubectl get nodes -l work_type=infra_core -o jsonpath='{.items[0].status.addresses[?(@.type=="InternalIP")].address}')

# Copy kubelet config
ssh -o StrictHostKeyChecking=no -i /home/ubuntu/.ssh/id_rsa ubuntu@${WORKER} \
  "sudo cat /var/lib/kubelet/config.yaml" > /var/work/tests/artifacts/4/kubelet-config.txt

# Extract cgroupDriver
ssh -o StrictHostKeyChecking=no -i /home/ubuntu/.ssh/id_rsa ubuntu@${WORKER} \
  "sudo grep cgroupDriver /var/lib/kubelet/config.yaml | awk '{print \$2}'" > /var/work/tests/artifacts/4/cgroup-driver.txt

# Extract containerd SystemdCgroup
ssh -o StrictHostKeyChecking=no -i /home/ubuntu/.ssh/id_rsa ubuntu@${WORKER} \
  "sudo grep SystemdCgroup /etc/containerd/config.toml | awk '{print \$3}'" > /var/work/tests/artifacts/4/containerd-cgroup.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/4/kubelet-config.txt
cat /var/work/tests/artifacts/4/cgroup-driver.txt
cat /var/work/tests/artifacts/4/containerd-cgroup.txt
```
