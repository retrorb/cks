# Task 17 — CNI detail: node CIDRs + Calico IPPool (4 pts)

## Problem
Save node podCIDRs and Calico IPPool YAML to artifact files.

## Solution

```bash
# Save node podCIDRs using jsonpath
kubectl get nodes -o jsonpath='{range .items[*]}{.metadata.name}{"\t"}{.spec.podCIDR}{"\n"}{end}' \
  > /var/work/tests/artifacts/17/node-cidrs.txt

# Save Calico IPPool YAML
kubectl get ippools -o yaml > /var/work/tests/artifacts/17/ippool.txt
# or if using calicoctl:
# calicoctl get ippools -o yaml > /var/work/tests/artifacts/17/ippool.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/17/node-cidrs.txt
cat /var/work/tests/artifacts/17/ippool.txt
```
