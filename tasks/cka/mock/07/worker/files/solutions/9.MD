# Task 9 — Helm template metrics-dash (5 pts)

## Problem
Template the metrics-dash chart with custom values and save the deployed image name.

## Solution

```bash
# Template with custom-values.yaml
helm template metrics-dash /var/work/helm/metrics-dash/ \
  -f /var/work/helm/metrics-dash/custom-values.yaml \
  -n helm-tpl-ns > /var/work/tests/artifacts/9/template.txt

# Get the deployed image (the release uses default values.yaml, not custom)
kubectl get deploy metrics-dash -n helm-tpl-ns \
  -o jsonpath='{.spec.template.spec.containers[0].image}' > /var/work/tests/artifacts/9/deployed-image.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/9/template.txt
cat /var/work/tests/artifacts/9/deployed-image.txt
helm list -n helm-tpl-ns
```
