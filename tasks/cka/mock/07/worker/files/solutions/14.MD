# Task 14 — Helm rollback broken-release (4 pts)

## Problem
Release `broken-release` at revision 3 has `replicas=0`. Rollback to revision 2 and save history.

## Solution

```bash
# Rollback to revision 2
helm rollback broken-release 2 -n helm-fix-ns

# Save history
helm history broken-release -n helm-fix-ns > /var/work/tests/artifacts/14/history.txt
```

Verify:
```bash
helm history broken-release -n helm-fix-ns
kubectl get deploy broken-release -n helm-fix-ns
cat /var/work/tests/artifacts/14/history.txt
```
