# Task 18 — NP+HPA compound: scaled-web (7 pts)

## Problem
Create an HPA and a NetworkPolicy for the `scaled-web` deployment.

## Solution

```bash
# Create HPA
cat <<EOF | kubectl apply -f -
apiVersion: autoscaling/v2
kind: HorizontalPodAutoscaler
metadata:
  name: scaled-web-hpa
  namespace: compound-scale-ns
spec:
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: scaled-web
  minReplicas: 2
  maxReplicas: 6
  metrics:
  - type: Resource
    resource:
      name: cpu
      target:
        type: Utilization
        averageUtilization: 80
EOF

# Create NetworkPolicy
cat <<EOF | kubectl apply -f -
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: scaled-web-policy
  namespace: compound-scale-ns
spec:
  podSelector:
    matchLabels:
      app: scaled-web
  policyTypes:
  - Ingress
  - Egress
  ingress:
  - from:
    - podSelector:
        matchLabels:
          role: loadbalancer
    ports:
    - protocol: TCP
      port: 80
  egress:
  - ports:
    - protocol: TCP
      port: 53
    - protocol: UDP
      port: 53
  - to:
    - podSelector:
        matchLabels:
          role: cache
    ports:
    - protocol: TCP
      port: 6379
EOF
```

Verify:
```bash
kubectl get hpa -n compound-scale-ns
kubectl get networkpolicy -n compound-scale-ns -o yaml
```
