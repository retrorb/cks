# Task 20 — Fix kubelet CA path on worker (6 pts)

## Problem
A worker node is NotReady because kubelet config references `ca-wrong.crt` instead of `ca.crt`.

## Solution

```bash
# Find the NotReady worker
kubectl get nodes

# SSH to the NotReady worker
ssh <worker-ip>

# Fix the kubelet config
sudo sed -i 's|ca-wrong.crt|ca.crt|g' /var/lib/kubelet/config.yaml

# Restart kubelet
sudo systemctl restart kubelet
```

Verify:
```bash
kubectl get nodes
sudo systemctl status kubelet
sudo grep clientCAFile /var/lib/kubelet/config.yaml
```
