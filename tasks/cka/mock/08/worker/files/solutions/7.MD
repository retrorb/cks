# Task 7 — Gateway multi-route (5 pts)

## Problem
Create 3 HTTPRoutes for the pre-existing Gateway `shared-gw`: path-based for /users and /orders, host-based for products.cluster1.local.

## Solution

```bash
# HTTPRoute for /users
cat <<EOF | kubectl apply -f -
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: users-route
  namespace: gw-multi-ns
spec:
  parentRefs:
  - name: shared-gw
  rules:
  - matches:
    - path:
        type: PathPrefix
        value: /users
    backendRefs:
    - name: users-svc
      port: 80
EOF

# HTTPRoute for /orders
cat <<EOF | kubectl apply -f -
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: orders-route
  namespace: gw-multi-ns
spec:
  parentRefs:
  - name: shared-gw
  rules:
  - matches:
    - path:
        type: PathPrefix
        value: /orders
    backendRefs:
    - name: orders-svc
      port: 80
EOF

# HTTPRoute for products.cluster1.local
cat <<EOF | kubectl apply -f -
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: products-route
  namespace: gw-multi-ns
spec:
  parentRefs:
  - name: shared-gw
  hostnames:
  - products.cluster1.local
  rules:
  - backendRefs:
    - name: products-svc
      port: 80
EOF
```

Verify:
```bash
kubectl get httproute -n gw-multi-ns
kubectl get gateway shared-gw -n gw-multi-ns
```
