# Task 19 — Gateway+CNI: secure-gw with TLS (5 pts)

## Problem
Create a Gateway with HTTP and HTTPS listeners, an HTTPRoute, and save CNI config.

## Solution

```bash
# Create Gateway with HTTP + HTTPS
cat <<EOF | kubectl apply -f -
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: secure-gw
  namespace: gw-secure-ns
spec:
  gatewayClassName: nginx-gw
  listeners:
  - name: http
    protocol: HTTP
    port: 80
  - name: https
    protocol: HTTPS
    port: 443
    tls:
      mode: Terminate
      certificateRefs:
      - name: secure-tls
EOF

# Create HTTPRoute
cat <<EOF | kubectl apply -f -
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: secure-route
  namespace: gw-secure-ns
spec:
  parentRefs:
  - name: secure-gw
  hostnames:
  - secure.cluster1.local
  rules:
  - backendRefs:
    - name: secure-backend-svc
      port: 80
EOF

# Save CNI config from master
MASTER=$(kubectl get nodes -l 'node-role.kubernetes.io/control-plane' -o jsonpath='{.items[0].status.addresses[?(@.type=="InternalIP")].address}')
ssh $MASTER "sudo cat /etc/cni/net.d/*.conflist 2>/dev/null || sudo cat /etc/cni/net.d/*.conf 2>/dev/null" > /var/work/tests/artifacts/19/cni-conf.txt
```

Verify:
```bash
kubectl get gateway secure-gw -n gw-secure-ns -o yaml
kubectl get httproute secure-route -n gw-secure-ns -o yaml
cat /var/work/tests/artifacts/19/cni-conf.txt
```
