# Task 6 — NetworkPolicy egress control (6 pts)

## Problem
Create a NetworkPolicy for `egress-pod` that denies all egress except to backend-ns port 8080, DNS 53, and CIDR 10.0.0.0/8 port 443.

## Solution

```bash
cat <<EOF | kubectl apply -f -
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: egress-control
  namespace: egress-ctl-ns
spec:
  podSelector:
    matchLabels:
      app: egress-pod
  policyTypes:
  - Egress
  egress:
  - to:
    - namespaceSelector:
        matchLabels:
          env: backend
    ports:
    - protocol: TCP
      port: 8080
  - ports:
    - protocol: TCP
      port: 53
    - protocol: UDP
      port: 53
  - to:
    - ipBlock:
        cidr: 10.0.0.0/8
    ports:
    - protocol: TCP
      port: 443
EOF
```

Verify:
```bash
kubectl get networkpolicy -n egress-ctl-ns -o yaml
```
