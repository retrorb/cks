# Task 1 — Helm: add repo, search, install ingress-nginx (4 pts)

## Problem
Add the ingress-nginx Helm repo, search for the chart, save output to artifact, and install a release with NodePort service type.

## Solution

```bash
# Add repo (already added by worker.sh, but ensure it exists)
helm repo add ingress-nginx https://kubernetes.github.io/ingress-nginx
helm repo update

# Search and save
helm search repo ingress-nginx > /var/work/tests/artifacts/1/search.txt

# Install release
helm install web-ingress ingress-nginx/ingress-nginx \
  -n helm-repo-ns \
  --set controller.service.type=NodePort
```

Verify:
```bash
helm list -n helm-repo-ns
cat /var/work/tests/artifacts/1/search.txt
```
