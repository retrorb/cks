# Task 15 — CNI inspect: node podCIDRs and CNI config (3 pts)

## Problem
Save node podCIDRs and CNI configuration from the master node.

## Solution

```bash
# Save node podCIDRs
kubectl get nodes -o jsonpath='{range .items[*]}{.metadata.name}{"\t"}{.spec.podCIDR}{"\n"}{end}' > /var/work/tests/artifacts/15/node-cidrs.txt

# Get CNI config from master
MASTER=$(kubectl get nodes -l 'node-role.kubernetes.io/control-plane' -o jsonpath='{.items[0].status.addresses[?(@.type=="InternalIP")].address}')
ssh $MASTER "sudo cat /etc/cni/net.d/*.conflist 2>/dev/null || sudo cat /etc/cni/net.d/*.conf 2>/dev/null" > /var/work/tests/artifacts/15/cni-conf.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/15/node-cidrs.txt
cat /var/work/tests/artifacts/15/cni-conf.txt
```
