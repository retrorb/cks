# Task 11 — Helm upgrade with prod values (4 pts)

## Problem
Install a Helm chart with defaults, template with prod values, then upgrade with prod values.

## Solution

```bash
# Install with defaults
helm install webapp-v1 /var/work/helm/webapp/ -n helm-compare-ns

# Template with prod values and save
helm template webapp-v1 /var/work/helm/webapp/ -f /var/work/helm/webapp/prod-values.yaml \
  > /var/work/tests/artifacts/11/prod-template.txt

# Upgrade with prod values
helm upgrade webapp-v1 /var/work/helm/webapp/ -n helm-compare-ns \
  -f /var/work/helm/webapp/prod-values.yaml
```

Verify:
```bash
helm list -n helm-compare-ns
helm get values webapp-v1 -n helm-compare-ns --all
cat /var/work/tests/artifacts/11/prod-template.txt
```
