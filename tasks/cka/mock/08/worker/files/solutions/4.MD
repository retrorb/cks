# Task 4 — Sidecar adapter pod (5 pts)

## Problem
Create a Pod with a main nginx container and a sidecar busybox that reads logs and writes adapted output.

## Solution

```bash
cat <<EOF | kubectl apply -f -
apiVersion: v1
kind: Pod
metadata:
  name: adapter-pod
  namespace: adapter-ns
spec:
  containers:
  - name: main
    image: nginx
    volumeMounts:
    - name: raw-logs
      mountPath: /var/log/nginx
  - name: adapter
    image: busybox
    command: ["/bin/sh", "-c"]
    args:
    - |
      while true; do
        if [ -f /var/log/nginx/access.log ]; then
          tail -f /var/log/nginx/access.log | while read line; do
            echo "[ADAPTED] \$line" >> /var/log/adapted/output.log
          done
        fi
        sleep 1
      done
    volumeMounts:
    - name: raw-logs
      mountPath: /var/log/nginx
    - name: adapted-logs
      mountPath: /var/log/adapted
  volumes:
  - name: raw-logs
    emptyDir: {}
  - name: adapted-logs
    emptyDir: {}
EOF
```

Verify:
```bash
kubectl get pod adapter-pod -n adapter-ns
kubectl describe pod adapter-pod -n adapter-ns
```
