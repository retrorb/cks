# Task 9 — Fix containerd disabled CRI plugin (6 pts)

## Problem
A worker node is NotReady because containerd has `disabled_plugins = ["cri"]` in its config.

## Solution

```bash
# Find the NotReady worker
kubectl get nodes

# SSH to the NotReady worker
ssh <worker-ip>

# Remove or comment out the disabled_plugins line
sudo sed -i '/disabled_plugins.*cri/d' /etc/containerd/config.toml

# Restart containerd and kubelet
sudo systemctl restart containerd
sudo systemctl restart kubelet
```

Verify:
```bash
kubectl get nodes
sudo systemctl status containerd
sudo systemctl status kubelet
```
