# Task 17 — Kubelet certs: cert expiry and journal (3 pts)

## Problem
Save the kubelet certificate expiry date and recent kubelet journal logs.

## Solution

```bash
MASTER=$(kubectl get nodes -l 'node-role.kubernetes.io/control-plane' -o jsonpath='{.items[0].status.addresses[?(@.type=="InternalIP")].address}')

# Save cert expiry
ssh $MASTER "sudo openssl x509 -enddate -noout -in /var/lib/kubelet/pki/kubelet.crt" > /var/work/tests/artifacts/17/cert-expiry.txt

# Save last 50 lines of kubelet journal
ssh $MASTER "sudo journalctl -u kubelet --no-pager -n 50" > /var/work/tests/artifacts/17/kubelet-journal.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/17/cert-expiry.txt
cat /var/work/tests/artifacts/17/kubelet-journal.txt
```
