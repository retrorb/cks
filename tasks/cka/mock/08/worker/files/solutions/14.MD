# Task 14 — Control plane inspect: apiserver flags (3 pts)

## Problem
Extract specific kube-apiserver flags and save to artifacts.

## Solution

```bash
# SSH to master or use kubectl to get the apiserver pod spec
MASTER=$(kubectl get nodes -l 'node-role.kubernetes.io/control-plane' -o jsonpath='{.items[0].status.addresses[?(@.type=="InternalIP")].address}')

# Get service-cluster-ip-range
ssh $MASTER "sudo cat /etc/kubernetes/manifests/kube-apiserver.yaml" | grep service-cluster-ip-range | awk -F= '{print $2}' > /var/work/tests/artifacts/14/svc-cidr.txt

# Get etcd-servers
ssh $MASTER "sudo cat /etc/kubernetes/manifests/kube-apiserver.yaml" | grep etcd-servers | awk -F= '{print $2}' > /var/work/tests/artifacts/14/etcd-url.txt

# Get authorization-mode
ssh $MASTER "sudo cat /etc/kubernetes/manifests/kube-apiserver.yaml" | grep authorization-mode | awk -F= '{print $2}' > /var/work/tests/artifacts/14/auth-mode.txt
```

Verify:
```bash
cat /var/work/tests/artifacts/14/svc-cidr.txt
cat /var/work/tests/artifacts/14/etcd-url.txt
cat /var/work/tests/artifacts/14/auth-mode.txt
```
