# Check current deployment

```
# Patch: change strategy to Recreate and update image
kubectl patch deployment batch-proc -n recreate-ns --type=json -p='[
  {"op":"replace","path":"/spec/strategy","value":{"type":"Recreate"}},
  {"op":"replace","path":"/spec/template/spec/containers/0/image","value":"nginx:alpine"}
]'
```
```
# Alternatively, edit directly:
# kubectl edit deployment batch-proc -n recreate-ns

```
# Verify

kubectl get deployment batch-proc -n recreate-ns -o jsonpath='{.spec.template.spec.containers[0].image}'
kubectl rollout status deployment/batch-proc -n recreate-ns
